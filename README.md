# NLW Node

#### This project stores a Node JS with Typescript app from Rocketseat course.

- You must have yarn installed →

```bash
$ yarn --version
1.22.10
```

- Make sure to install all Project/Dev dependencies.

```bash
$ yarn install
```

- Then, you need to run the server →

```bash
$ yarn dev
```
